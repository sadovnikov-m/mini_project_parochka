## Опис ідеї та обґрунтування вибору фінансового інструменту

**Обраний актив:** Bitcoin ('BTC-USD')

### Обґрунтування вибору:
* **Економічний інтерес:** Bitcoin є ключовим активом ринку криптовалют і все частіше розглядається інвесторами як «цифрове золото» та засіб хеджування проти інфляції.
* **Технічна відповідність стратегії:** Завдяки своїй високій волатильності та наявності тривалих виражених трендів, Bitcoin є ідеальним об'єктом для тестування трендових стратегій, зокрема **Moving Average Crossover**.
* **Якість даних:** Ринок криптовалют працює 24/7, що дало змогу отримати безперервний та чистий часовий ряд без пропусків на вихідні дні.

## Опис процесу аналізу та використані інструменти

Для реалізації проєкту було використано мову програмування **Python 3** та такі бібліотеки:
* `yfinance` — для отримання історичних ринкових даних BTC-USD через API.
* `pandas` & `numpy` — для очищення, структурування даних у DataFrame та проведення векторних розрахунків.
* `matplotlib` — для побудови графіків ціни, ковзних середніх та точок входу/виходу з позиції.

### Торгова логіка:
* **Коротка SMA (30 днів):** швидка лінія, яка реагує на поточні зміни ціни.
* **Довга SMA (60 днів):** повільна лінія, яка визначає основний тренд.
* **Сигнал КУПІВЛІ (Buy):** коротка SMA перетинає довгу SMA знизу вгору.
* **Сигнал ПРОДАЖУ (Sell):** коротка SMA перетинає довгу SMA згори вниз.
